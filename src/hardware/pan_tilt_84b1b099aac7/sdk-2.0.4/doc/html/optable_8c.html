struct cpi_opent cpi_optable[]

Opcode lookup table.

The SDK ships with this table containing every allowable opcode. As long as the table is not trimmed, lookup is O(1) via direct indexing off the integer value of cpi_opcode. If the table is trimmed, then CPI_CONFIG_USE_OP_LKUP must be defined. This definition builds a binary search lookup function as an alternative lookup method.
