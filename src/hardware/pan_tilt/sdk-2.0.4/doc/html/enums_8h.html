Detailed Description

Common enumerations and defines.

All enumerations are wire encoded as integers. Because the C specification allows the compiler to decide if an enum is representated as a char or as an int, all enumerations must be explicitly cast to int. For clarity's sake, cpi_enum is a typedef to int.

Example: 

 cpi_ptcmd(cer, &status, OP_EXEC_MODE_SET, (cpi_enum)CPI_IMMEDIATE_MODE);
